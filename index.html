About the Project

The idea for FileNames WordCount sparked from a personal challenge I faced while organizing my personal work. As a translator, I found myself with thousands of documents. I needed a tool to analyze these file names, hoping to make it faster and easier to add the documents and their WordCount in my excel sheet.

What started as a simple script to count words in my documents file names evolved into a more robust tool that could scan entire directories, handle various file types, and export data for further analysis. The project took shape over six weeks, from initial concept to the feature-complete version you see today.

This project is part of my portfolio for the Holberton School curriculum, showcasing my skills in Python programming, file system operations, and data analysis.

The Story Behind Document Word Counter

As a professional translator, my work involves handling numerous text files throughout the month. At the end of each month, I'm required to submit an Excel sheet detailing all the files I've worked on and their respective word counts. This task, while seemingly simple, became increasingly challenging as my workload grew. I vividly remember one particularly hectic month. It was nearing the deadline, and I found myself surrounded by a sea of documents - Word files, PowerPoint presentations, and Excel sheets. The clock was ticking, and I was manually opening each file, counting words, and meticulously recording the data in my report. The process was not only time-consuming but also prone to errors, especially when fatigue set in during those late-night sessions. As I sat there, bleary-eyed, copying and pasting numbers, a thought struck me: "Why am I doing this manually? Couldn't I write a Python script to do this for me?" This moment of frustration became the spark that ignited the Document Word Counter project. This is really happened so you can realize how important this project is for me and for people like me

Feature Walkthrough

Document Word Counter is designed to be simple yet powerful. Here's a quick overview of how to use the main features:

  1. Directory Scanning: Simply provide the path to the directory you want to analyze. The script will automatically scan all files in the directory.
  2. Word Counting: The script intelligently separates words in file names.
  3. CSV Export: Results are neatly organized into a CSV file, making it easy to import into spreadsheet software for further analysis or visualization.

For a detailed guide on how to use FileNames WordCount, check out our README on GitHub.
